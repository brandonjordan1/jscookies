    Easily Create Cookies with JavaScript

    Project maintained by brandonjordan1 Hosted on GitHub Pages — Theme by mattgraham

    Usage

    Download 'jsCookies.js', and put this in the <head> of your code:

    <script type="text/javascript" src="../jscookies.js"></script>

    Get Cookie Value

    getCookie("name");

    Create Cookie & Set Value

    setCookie("name","value",365);

    name Name of the cookie
    value Value of the cookie
    365 Number of days till cookie expires

    Delete Cookie

    eraseCookie("name");
